=== source_code ===

All synthesizable Verilog/SystemVerilog source files for the Pong project are placed here. Each file corresponds to a specific hardware module (e.g., game logic, VGA render, input controllers).

本目录存放Pong项目的所有可综合Verilog/SystemVerilog源代码，每个文件对应一个具体的硬件模块（如游戏逻辑、VGA图像生成、输入控制器等）。

=== Special Note: clk_wiz.v (Clock Module) ===

clk_wiz.v is a wrapper that instantiates the clk_wiz_0 core generated by Vivado's IP Catalog (Clocking Wizard). It is NOT a standalone plain-Verilog module and will not synthesize or simulate outside a Vivado project with the proper IP configured.

Collaborator notes:
- If re-creating the Vivado project from scratch, use IP Catalog -> Clocking Wizard to generate an MMCM with input 100 MHz, output 25.175 MHz, and name the IP instance "clk_wiz_0".
- Alternatively, implement an equivalent clock divider / MMCM wrapper manually.
- clk_wiz.v has no corresponding testbench because the IP core is vendor-provided and not intended for RTL simulation in isolation.
- vgac.v is a course-provided VGA timing module; it is a plain Verilog file and can be freely simulated.

=== 特别说明：clk_wiz.v（时钟模块） ===

clk_wiz.v 是 Vivado IP Catalog 中 Clocking Wizard 生成的 clk_wiz_0 核心的包装器（wrapper），并非独立的纯 Verilog 模块。在缺少对应 IP 配置的 Vivado 项目外，该文件无法直接综合或仿真。

协作者须知：
- 若从头重建 Vivado 项目，请通过 IP Catalog -> Clocking Wizard 生成 MMCM，输入 100 MHz、输出 25.175 MHz，并将 IP 实例命名为 "clk_wiz_0"。
- 也可自行实现等效的时钟分频 / MMCM 包装模块。
- clk_wiz.v 没有对应的仿真测试文件，因为该 IP 核由厂商提供，无需单独进行 RTL 仿真。
- vgac.v 为课程提供的 VGA 时序模块，是纯 Verilog 文件，可以自由仿真。
